לחץ "Create API Key" → שם: expertiza → Create
העתק את ה-Key — הוא מתחיל כך:
$2a$10$xxxxxxxxxxxxxxxxxxxxxxxx...
3
צור Bin לאחסון הנתונים

במקום הטקסט הכנס: {}
לחץ "Create"
העתק את ה-BIN ID (מספר שנראה כך):
64a8f2c3e3267a27ec123456
4
הכנס את הפרטים כאן
